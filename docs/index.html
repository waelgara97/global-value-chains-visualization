How to View the Full Interactive Visualization

This is a static preview of the visualization. To experience the full interactive version:

Option 1: View directly from GitHub

  1. Go to the repository: https://github.com/waelgara97/global-value-chains-visualization
  2. Files are in the src directory

Option 2: Run locally

  1. Clone the repository: git clone https://github.com/waelgara97/global-value-chains-visualization.git
  2. Navigate to the folder: cd global-value-chains-visualization
  3. Install dependencies: npm install
  4. Start the development server: npm start
  5. Open your browser at: http://localhost:3000
